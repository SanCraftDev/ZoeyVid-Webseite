Mod-Installer

With this mod installer, you can install All Optifine, Fabric, Forge and Iris versions and ESSENTIAL, which are working!
And you can also install my Client Modpacks for 1.19.x and also download some texture packs!
In addition, you can save mods in up to 8 profiles and start Minecraft via an extra script and directly select the profile that should be loaded.
A Minecraft Modpack is supported at least until the next version is stable!
Modpacks always run on the latest .x version (for example not on 1.19.1 but alway on the latest Version, on 1.19.2)

You can find the latest download here
Feel free to leave feedback, bugs and suggestions in the issues!

Requirements:

  • Windows 10 or 11 Device with an Internet connection, a Microsoft Account connected to the Microsoft Store and administrator permissions on your device
  • Winget - should be preinstalled on most Windows 10 systems and every Windows 11 system

If you want to use a Controller in Minecraft, install Steam and Setup a Desktop Controller Configuration, it will work in Minecraft
